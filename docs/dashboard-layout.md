# Dashboard Layout & UX

## 1. Display Context

- Large TV screen
- Viewed from ~3 metres
- Non‑interactive
- Auto‑rotating slides

## 2. Visual Style

- Inspired by Government of Canada branding
- Not a rigid GC form layout
- Clean, modern, bold
- High contrast
- Generous spacing
- Minimal text

## 3. Typography

- Large numeric displays
- Sans-serif fonts
- Minimum 48px for key metrics
- Minimum 32px for labels

## 4. Slide Timing

- ~10 seconds per slide
- English → French transition
- Smooth fade or slide animation

## 5. Slide Structure

- Title (EN/FR)
- Key metric(s)
- Chart
- Short insight sentence
- Footer with source
