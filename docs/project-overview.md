# Canadian Business Dashboard — Project Overview

## 1. Purpose
The Canadian Business Dashboard is a public‑facing, non‑interactive data visualization designed to help government staff and visitors quickly understand trends in Canadian business activity. It will be displayed on a large TV screen in lobbies, waiting areas, and informal walkthrough spaces.

The goal is to transform complex federal datasets into clear, accessible, visually engaging insights that can be understood at a glance from approximately 3 metres away.

## 2. Audience
- Government of Canada staff  
- Visitors and stakeholders  
- Public audiences in federal buildings  
- Individuals with no data background  

## 3. Key Features
- Bilingual (English + French)  
- Non‑interactive, auto‑rotating slides  
- High‑contrast, large‑type visualizations  
- Clean, modern design inspired by GC branding  
- Data sourced exclusively from the Open Government Portal  
- Trends from 2016 to the most recent available year  
- Provincial, territorial, and rural/urban breakdowns  

## 4. High-Level Deliverables
- A React + Vite dashboard deployed on GitHub Pages  
- Cleaned and structured JSON datasets  
- Visualizations for:
  - New business creation (with/without employees)
  - Rural business profiles
  - Sector, city, and province comparisons
  - Fun‑fact insights combining data + contextual internet facts
  - Pre‑/post‑COVID comparisons and projections

## 5. Success Criteria
- Legible from 3 metres  
- Smooth bilingual transitions  
- Accurate, reproducible data transformations  
- Visually consistent slide sequence  
- Zero user interaction required  
- Fast load time on GitHub Pages
