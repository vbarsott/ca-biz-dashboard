# Risks & Constraints

## 1. Data Risks

- Incomplete rural/urban classifications
- Missing values in sector datasets
- Yearly updates may change trends

## 2. Technical Risks

- GitHub Pages caching
- Large JSON files affecting load time
- Chart readability on large screens

## 3. UX Risks

- Overly dense charts
- Insufficient contrast
- Text too small for 3‑metre viewing

## 4. Constraints

- No user interaction
- No backend
- Must run fully static
- Must support bilingual display
